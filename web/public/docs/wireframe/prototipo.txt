Prótotipo de telas:

LOGIN - OK

tela com email e senha, checkbox de lembrar senha

-

PERFIL - OK

Exibe dados pessoais, já em edição.
nome , email e senha
	senha - se deixar em branco não altera.

-

CLIENTES - OK

Listagem clientes, com nome
AÇÕES - editar / excluir
	Editar - exibe a lista de contatos deste cliente, nome email, funcao, observacao, botao de adicionar, editar e excluir contato

-

INDEX - PROJETOS - OK

Lista últimos projetos, com data inicial, nome, descrição , cliente - OK
AÇÕES - editar / excluir - OK
	Editar - mostra em cima a edição dos campos Mostra os REQUISITOS - OK
Botão adicionar novo projeto - OK

-

REQUISITOS - OK

Listagem de requisitos, separados por feitos e não feitos
Botão adicionar novo requisito
AÇÕES - editar / excluir
	Editar - edita codigo, nome, prioridade, estabilidade, tipo de requisito, impacto na arquitetura, descricao, requisito pai, contato do cliente q pediu o requisito.
		terá um botão para finalizar o requisito, dar como pronto.

-

SOBRE - OK

Modal com informações sobre o desenvolvimento

-

SAIR - OK

logout do sistema, redireciona para tela de login.
